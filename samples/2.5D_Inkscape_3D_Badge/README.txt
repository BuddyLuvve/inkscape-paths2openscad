# Summary

This medallion is another 3D object (after http://www.thingiverse.com/thing:2215995 andhttp://www.thingiverse.com/thing:2215995 ) done entirley within inkscape. Technical details see https://github.com/fablabnbg/inkscape-paths2openscad

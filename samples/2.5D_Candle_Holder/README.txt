# Summary

This is my first 3D object created with inkscape.
Being a 2D tool, inkscape cannot do real 3D objects. 

With the help of an extension, we can assign height values to individual objects.

See http://www.thingiverse.com/thing:25036 for the original extension by Dan Newman.
See https://github.com/fablabnbg/inkscape-paths2openscad/releases for an update that integrates a simplified 3D printing workflow via openscad and cura directly with inkscape. There is python sources and DEB package.
